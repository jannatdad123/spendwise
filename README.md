# SpendWise - Expense Tracker App

A Flutter-based expense tracker mobile application developed as a university semester project for managing daily expenses with local data storage and categorized expense tracking.

## Features
- Add daily expenses
- Edit and delete expense entries
- Categorize expenses
- Track total spending
- Local data persistence using Hive
- Clean and responsive UI

## Technologies Used
- Flutter
- Dart
- Hive Database
- VS Code

## Project Type
University Semester Project
